検証者 · VERIFIER

① Digital Credentials API（ネイティブウォレット）

ブラウザ内蔵 navigator.credentials.get() で OS ウォレットに要求（Chrome 141+ / Android DC API 対応端末が必要）。

準備完了。ユーザー操作（ボタン）起点で DC API を呼び出します。








  
② Web ウォレットに要求（OID4VP HTTPS リダイレクト）

  

    DC API 非対応環境向け。Verifier がリクエストを生成し、Web ウォレットへリダイレクトします。
    ウォレットが提示し、結果を direct_post.jwt で返します。
  

  

    
    
  







  
③ 検証コンソール（要求ビルダー＋自己完結検証）

  

    16 種 × {mdoc / SD-JWT}・項目選択・選択的開示・プロトコル選択・DCQL JSON を確認できるフルコンソール。
    発行者からテスト用クレデンシャルを取得し、その場で提示・検証まで実行します。
  

  

    /verifier（検証コンソールを開く）
